# 🛰️ Voyager 1 Tracker

A simple 3D space dashboard that tracks the Voyager 1 probe. It uses HTML and CSS for the text stats layout on the left side, and Three.js to render a spinning planet, an orbital ring, and a floating probe capsule on the right side.

Built for the Hack Club Polygon YSWS.

**Live Link:** https://voyager-telemetry.vercel.app
